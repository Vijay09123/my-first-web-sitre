🌐 What is Web Development?

Web development means building websites and web applications. It's like building a house:

🏠 House Analogy:

  • HTML = Foundation & Walls (Structure)
  • CSS = Paint & Decorations (Style)
  • JavaScript = Electricity & Appliances (Function)
💡 Tip: Start with HTML first, then CSS, then JavaScript!

📄 HTML - The Structure

HTML (HyperText Markup Language) defines the structure of a webpage. Think of it as the skeleton of your website.

Basic Tags:

<!-- This is a comment --> <h1> Heading </h1> <p> Paragraph </p> <button> Click Me </button> <a href="google.com"> Link </a> <img src="photo.jpg" />

🎨 CSS - The Style

CSS (Cascading Style Sheets) makes your website look beautiful! It handles colors, fonts, layout, and animations.

Basic Properties:

selector { color: red; font-size: 20px; background: blue; margin: 10px; padding: 20px; border-radius: 10px; }

Selector Examples:

  • h1 - Selects all heading tags
  • .class - Selects elements with a class
  • #id - Selects element with an ID

⚡ JavaScript - The Function

JavaScript adds interactivity to your website! It can respond to clicks, change content, and do calculations.

Basic Syntax:

// Variables let name = "Prachi"; const age = 18; // Function function sayHello() { console.log("Hello!"); } // Event button.addEventListener("click", sayHello);

🚀 Mini Project: Color Changer

Let's build a simple project - a button that changes the background color!

<button id="changeBtn">Change Color</button> <script> const colors = ["red", "blue", "green", "pink", "orange"]; let index = 0; document.getElementById("changeBtn").addEventListener("click", function() { document.body.style.background = colors[index]; index = (index + 1) % colors.length; }); </script>
